# 亚马逊商家，与自己化敌为友

> 2020年即将走向尾声，您在备战和冲刺亚马逊Q4旺季的同时，是否发现销售瓶颈、乐天雅虎冲击、爆款太难呢？

日本市场体量庞大，消费力高，近年来成为中国跨境电商卖家的重点关注对象，加之招商经理的推广，国内卖家大量涌进日本亚马逊市场，导致竞争加剧，利润缩水。 疫情原因，日本另外两大巨头“乐天”、“雅虎”频频出击，促销活动接二连三，也大大影响了亚马逊商家的销售额。 我们深耕日本市场十多年，力图以“公司代运营”、 “仓储配送代运营” 、“店铺代运营”等一条龙服务帮助亚马逊商家在乐天雅虎平台扬帆起航。

### 内部分析-亚马逊商家的喜与忧

**日贸EC市场的不断增长**

日本是全球第三大经济体，国民生产总值位居世界第三，人口1.27亿，互联网普及率94%，人均网购消费金额1164美金，数据显示，预计到2022年，日本电商市场价值将超过1130亿美元。而随着中国产品质量的不断提高，历代日贸前辈的努力，中国制造已经渗入到了日本生活的方方面面，即使是挑剔的日本买家，中国制的产品销量依旧不断上涨。根据2019年9月の日本流通産業新聞报道Amazonジャパン1兆5,280亿日元居第二。

**亚马逊商家的困扰**

* 注册审核越来越严格，对于想多账号运营的商家而言入驻越来越困难。
* 国人商家越来越多，而亚马逊的出品形式促使不断压低价格，利润受损。
* FBA入库限制越来越多，按件计费的模式让配送成本始终无法压缩。
* 商品即使大卖也不能积累出品牌粉丝，推广新产品和也需从零开始。
* 乐天雅虎平台如火如荼，导致销量波动，却只能被动等待，无从下手。
